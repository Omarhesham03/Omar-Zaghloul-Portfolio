# Lecture Plan Generator

## What This Skill Does
Takes a course syllabus or topic list and generates detailed per-week/per-session lecture plans with learning objectives, content outline, activities, and assigned readings.

## Per-Session Template
```
WEEK [N] — SESSION [N]: [Topic Title]
Date: [Date]
Duration: [Minutes]

LEARNING OBJECTIVES:
By the end of this session, students will be able to:
- [Verb from Bloom's] [specific outcome]
- [Verb] [outcome]

PRE-CLASS:
- Read: [Chapter/paper/resource]
- Watch: [Video, if flipped classroom]
- Prepare: [Pre-class exercise]

SESSION PLAN:
[0:00-0:10] Recap + Q&A on previous session
[0:10-0:35] Lecture: [Sub-topic 1]
  - Key concepts: [list]
  - Examples: [list]
[0:35-0:50] Activity: [Hands-on / discussion / problem-solving]
[0:50-0:60] Wrap-up + preview next session

MATERIALS NEEDED:
- Slides: [filename]
- Code: [repository/file]
- Hardware: [if applicable]

ASSESSMENT CONNECTION:
This session prepares students for: [HW N / Lab N / Exam topic]
```
