# Blog Post and Article Writer
## Structure
```
TITLE: [SEO-optimized, <60 chars]
META DESCRIPTION: [150-160 chars, includes primary keyword]
TARGET KEYWORD: [Primary keyword]
WORD COUNT: [Target]

OUTLINE:
H1: [Title]
  Introduction (hook + thesis + preview)
H2: [Section 1]
  [Key points]
H2: [Section 2]
  [Key points]
H2: [Section 3]
  [Key points]
H2: Conclusion
  [Summary + CTA]
```
## SEO Rules
- Primary keyword in: title, H1, first 100 words, meta description, URL slug
- Secondary keywords distributed naturally
- Internal links: 2-3 to related content
- External links: 1-2 to authoritative sources
- Images: alt text with keywords (naturally)
- Readability: short paragraphs, subheadings every 200-300 words
