# Conference Submission Manager

## What This Skill Does
Manages the logistics of academic paper submissions: deadline tracking, template compliance, cover letter generation, and submission checklist verification.

## Submission Checklist
- [ ] Paper formatted to venue template (IEEE, ACM, Springer LNCS)
- [ ] Page limit respected (excluding references if allowed)
- [ ] Abstract within word limit
- [ ] All figures legible at print resolution (300 DPI minimum)
- [ ] References complete (no "[?]" or missing fields)
- [ ] Author information matches submission system
- [ ] Supplementary materials prepared (code, data, video)
- [ ] Conflicts of interest declared
- [ ] Cover letter drafted (if required)
- [ ] PDF generated from LaTeX (not Word conversion)
- [ ] Blind review compliance (no author names in paper if double-blind)

## Cover Letter Template
```
Dear [Editor/Program Chairs],

We submit our paper titled "[Title]" for consideration at [Venue].

This paper presents [1-sentence contribution]. Our key results show [1-sentence highlight]. This work is relevant to [Venue] because [alignment with scope].

The paper has not been submitted elsewhere and all authors have approved this submission.

Sincerely,
[Corresponding author]
```
