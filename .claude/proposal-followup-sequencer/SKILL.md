# Proposal Follow-Up Sequencer
## Sequence
```
Day 0: Proposal sent
Day 2: Follow-up #1 — "Checking in"
  - Confirm receipt
  - Offer to walk through the proposal on a call
  - Add one new piece of value (insight, case study)

Day 5: Follow-up #2 — "New angle"
  - Reference a relevant industry trend or their competitor
  - Reframe key benefit from proposal
  - Offer reduced-scope option if budget is the blocker

Day 10: Follow-up #3 — "Direct ask"
  - "Is this still a priority for Q[X]?"
  - Ask about timeline and decision process
  - Suggest a specific next step

Day 15: Follow-up #4 — "Break-up email"
  - "I don't want to keep bothering you"
  - Leave door open: "If timing changes, I'm here"
  - Remove from active sequence after this
```
## Rules
- Never send more than 4 follow-ups on a single proposal
- Each follow-up must add new value (not just "checking in")
- Match follow-up channel to how the relationship started
- If they respond at any point, exit the sequence and have a real conversation
