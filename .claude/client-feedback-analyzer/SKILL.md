# Client Feedback Collector and Analyzer
## Survey Template
```
PROJECT COMPLETION FEEDBACK
===========================
Project: [Name]
Client: [Name]

1. Overall satisfaction (1-10): ___
2. Communication quality (1-10): ___
3. Deliverable quality (1-10): ___
4. Timeline adherence (1-10): ___
5. Value for money (1-10): ___
6. What went well? [Open text]
7. What could improve? [Open text]
8. Would you recommend us? (1-10 NPS): ___
9. Would you work with us again? [Yes/No/Maybe]
10. Any other comments? [Open text]
```
## Analysis Output
```
CLIENT FEEDBACK ANALYSIS
========================
Response count: [N]
Average NPS: [Score] — [Promoter/Passive/Detractor]
Highest rated: [Category] ([Score])
Lowest rated: [Category] ([Score])

THEMES FROM OPEN RESPONSES:
Positive: [Theme 1], [Theme 2]
Improvement: [Theme 1], [Theme 2]

ACTION ITEMS:
1. [Specific improvement] — Owner: [Who] — Deadline: [When]
2. [...]
```
