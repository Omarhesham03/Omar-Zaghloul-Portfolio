# Social Media Bio and Profile Optimizer
## LinkedIn
```
HEADLINE: [Role] | [Specialty] | [Value proposition] (max 220 chars)
  Example: "MSc Computer Engineering | AI & Automation | Building intelligent systems for businesses"

ABOUT (2000 char max):
  Paragraph 1: What you do and who you help
  Paragraph 2: Key achievements/expertise
  Paragraph 3: What you're working on now
  CTA: How to reach you

EXPERIENCE: Achievement-focused bullets (not job descriptions)
SKILLS: Top 3 pinned = most important to your brand
FEATURED: Pin best work (projects, articles, media)
```
## GitHub
```
README.md profile:
  - One-line intro
  - Current focus
  - Tech stack badges
  - Pinned repos (6 best projects)
  - Contact/social links
```
## Twitter/X
```
BIO (160 chars): [Role] + [Interest/Niche] + [Personality hook]
PINNED TWEET: Best-performing or most representative tweet
```
