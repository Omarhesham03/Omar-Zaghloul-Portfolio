# Video Script Writer
## Script Template
```
VIDEO: [Title]
Duration: [Target minutes]
Type: [Tutorial / Explainer / Review / Vlog]

HOOK (0:00-0:30):
[Opening line that creates curiosity or states the problem]
[Why they should keep watching]

INTRO (0:30-1:00):
[Brief intro — who you are, what they'll learn]
[Subscribe CTA if YouTube]

BODY:
Section 1 (1:00-X:00): [Topic]
  SCRIPT: [What to say]
  VISUAL: [Screen recording / B-roll / graphic]
  KEY POINT: [Main takeaway]

Section 2 (X:00-Y:00): [Topic]
  SCRIPT: [What to say]
  VISUAL: [Visual direction]
  KEY POINT: [Main takeaway]

CONCLUSION (Last 30-60s):
  [Recap key points]
  [CTA: Subscribe, comment, next video]
  [End screen suggestions]

THUMBNAIL TEXT: [2-4 words on thumbnail]
DESCRIPTION: [SEO-optimized description with timestamps]
TAGS: [10-15 relevant tags]
```
