# Lab Report Formatter

## What This Skill Does
Formats experimental lab reports following academic conventions with proper structure, data tables, error analysis, and conclusions.

## Structure
```
TITLE
AUTHOR(S) AND DATE
ABSTRACT (100-200 words)
1. INTRODUCTION (objective, background, hypothesis)
2. MATERIALS AND METHODS (reproducible detail)
3. RESULTS (data tables, figures, observations — no interpretation)
4. DISCUSSION (interpretation, error analysis, comparison to theory)
5. CONCLUSION (did results support hypothesis?)
REFERENCES
APPENDIX (raw data, calculations)
```

## Rules
- Results section: present data, never interpret
- Discussion section: interpret data, explain anomalies
- Every figure/table must be referenced in text
- Error bars or uncertainty on all measurements
- Significant figures consistent with instrument precision
