# Habit and Goal Tracker
## Goal Structuring
```
GOAL: [What you want to achieve]
WHY: [Motivation — connects to values]
DEADLINE: [Target date]
METRIC: [How you'll know you achieved it]

MILESTONES:
| Milestone | Target Date | Metric | Status |
|-----------|------------|--------|--------|
| [M1] | [Date] | [Measurable] | [Not started/In progress/Done] |
| [M2] | [Date] | [Measurable] | |
| [M3] | [Date] | [Measurable] | |

HABITS (daily/weekly actions that drive the goal):
| Habit | Frequency | Streak | Last Done |
|-------|-----------|--------|-----------|
| [Habit] | [Daily/3x week] | [N days] | [Date] |

WEEKLY CHECK-IN:
- Progress toward next milestone: [On track / Behind / Ahead]
- Blockers: [What's in the way]
- Adjustment needed: [Yes/No — what changes]
```
