# Landing Page Copy Generator
## Page Structure
```
1. HERO SECTION
   Headline: [Primary value proposition — 6-12 words]
   Subheadline: [Supporting detail — 15-25 words]
   CTA: [Action button text — 2-5 words]
   Visual: [Hero image/video description]

2. PROBLEM SECTION
   Header: [Name the pain]
   Body: [3 pain points the audience recognizes]

3. SOLUTION SECTION
   Header: [How you solve it]
   Features: [3-4 key features with icons and 1-sentence descriptions]

4. SOCIAL PROOF
   - Testimonials (2-3 with name, title, photo)
   - Logos (client/partner logos)
   - Stats ("500+ projects delivered", "98% satisfaction")

5. HOW IT WORKS
   Step 1: [Simple step]
   Step 2: [Simple step]
   Step 3: [Result]

6. PRICING / CTA SECTION
   [Pricing table or final CTA with urgency]

7. FAQ
   [5-7 common objections answered]
```
## Headline Formulas
- [Outcome] without [Pain]: "Professional Websites Without the Agency Markup"
- [Number] [Benefit]: "3x Faster Project Delivery"
- Question: "Still Managing Projects in Spreadsheets?"
- How to: "How [Vertical] Companies Automate Their Operations"
