# Portfolio Case Study Writer
## Structure
```
CASE STUDY: [Project Name]
===========================
Client: [Name / Industry]
Duration: [Timeline]
Services: [What was provided]

THE CHALLENGE:
[2-3 sentences: what problem the client faced]

THE SOLUTION:
[3-4 sentences: what was built/delivered and key design decisions]

TECH STACK:
[List of technologies with brief justification for each]

KEY FEATURES:
• [Feature 1]: [What it does + user benefit]
• [Feature 2]: [What it does + user benefit]

RESULTS:
• [Metric 1]: [Before → After or specific improvement]
• [Metric 2]: [Quantified outcome]
• [Metric 3]: [Business impact]

CLIENT TESTIMONIAL:
"[Quote or template for client to fill]" — [Name, Title]

LESSONS LEARNED:
[1-2 sentences on what was learned, applicable to future projects]
```
