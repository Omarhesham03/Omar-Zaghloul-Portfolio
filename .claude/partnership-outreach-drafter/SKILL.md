# Partnership and Referral Outreach Drafter
## Templates

### Partnership Proposal
```
Subject: Partnership Opportunity — [Mutual benefit in 5 words]

Hi [Name],

I run [Company] — we [what you do] for [who you serve].

I noticed [Company] serves a similar audience with [their service]. Our services complement each other: [specific non-competing synergy].

I'm thinking:
- We refer clients who need [their service] to you
- You refer clients who need [your service] to us
- [Optional: co-create content / joint webinar / bundled offering]

No formal commitment needed — just a mutual introduction agreement to start.

Worth a 15-minute call to explore this?

Best,
[Name]
```

### Referral Program Pitch
```
REFERRAL PROGRAM: [Agency Name]
================================
Commission: [X]% of first project value (or flat fee of $[X])
Payment: Within 30 days of client payment
Tracking: Unique referral code or direct introduction
Duration: Lifetime on first project (or [X] months)
Exclusions: Existing leads already in pipeline
```
